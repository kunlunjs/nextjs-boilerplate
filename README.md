Next.js基础模板说明

## 初次使用

进入开发模式

```bash
npm run dev
# or
yarn dev
```

构建

```bash
npm run build
# or
yarn build
```

进入生产模式(必须在构建之后运行)


```bash
npm start
# or
yarn start
```
