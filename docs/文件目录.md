# 文件目录说明

```
/
├── .gitignore                gitignore文件
├── .eslintrc.js              eslint配置文件
├── .prettierrc.js            prettier配置文件
├── .husky                    husky hooks目录
├── components                组件目录
├── docs                      文档目录
├── interfaces                全局接口目录
├── locales                   国际化资源目录
├── {module}                  模块目录                                      
├── package.json              package.json文件
├── pages                     next.js默认路由目录，包含页面以及_app, _document等文件                     
├── public                    公共资源目录（包含图片，favicon以及静态资源）
├── styles                    全局样式目录
├── tsconfig.json             ts配置文件
├── utils                     全局工具方法目录
├── commitlint.config         commitlint配置文件
├── lint-staged.config.js     lint-staged配置文件
└── README.md                 项目说明文件                              
```
